### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users, teachers, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Announcement Information
5. Knowledge Base
6. Experimental Courses
7. Laboratory Information
8. Personal Center
9. My Collection
10. Change Password
11. Laboratory Reservation Management
12. My Collection Management



**Teacher Functions:**
1. Login
2. Register
3. Change Password
4. Personal Center
5. Announcement Information
6. Knowledge Base Management
7. Experimental Course Management
8. Experimental Reservation Management


**Administrator Functions:**
1. Login
2. Change Password
3. Personal Information
4. Student Management
5. Teacher Management
6. Announcement Information Management
7. Knowledge Base Management
8. Experimental Course Management
9. Laboratory Information Management
10. Laboratory Reservation Management
11. Experimental Equipment Management
12. Purchase Record Management
13. Maintenance Record Management
14. Carousel Image Management
